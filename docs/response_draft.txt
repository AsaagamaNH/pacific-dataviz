Ringkasan Proyek Infografis

Tujuan:
- Menunjukkan ketidakadilan iklim (climate justice) untuk negara‑negara kepulauan Pasifik yang berkontribusi sangat sedikit pada emisi gas rumah kaca, namun menjadi korban utama bencana iklim.

Data:
- 6 440 baris data gabungan (emisi, kenaikan permukaan laut, dampak bencana, kerugian ekonomi).
- Sekitar 54 % nilai kosong, diatasi dengan Hierarchical Mean Imputation (tahap 1: rata‑rata tahun yang sama; tahap 2: rata‑rata indikator seluruh tahun; tahap 3: 0).

Pengolahan:
- Script merge_and_impute.py menyiapkan grid lengkap, mengisi nilai kosong, mengekspor ke data_final_imputed.json untuk visualisasi.

Visualisasi (React + D3):
1. Babak 1: The Paradox – Emisi per kapita (waffle atau sorted horizontal bar chart).
2. Babak 2: The Silent Threat – Kenaikan permukaan laut (radial line atau area chart, warna neon cyan).
3. Babak 3: The Human & Economic Toll – Korban & kerugian (proportional‑symbol map + treemap, warna coral red).
4. Babak 4: Bridging the Data Gaps – Transparansi imputasi (heat‑map matrix).

Desain UI:
- Tema dark navy (#0a192f) dengan aksen neon cyan (#00f5d4) dan coral red (#f25c54).
- Font judul: Space Grotesk atau Outfit. Font paragraf: Inter atau Sora.

Struktur Repo:
Projek Infografis/
├─ merge_and_impute.py      # preprocessing & imputasi
├─ data_final_imputed.json # dataset bersih
├─ dataviz-react/           # front‑end React + D3
│   ├─ src/App.jsx
│   ├─ src/components/D3Charts.jsx
│   └─ index.html
└─ README.md                # dokumentasi ini

Cara Menjalankan:
1. pip install pandas
2. python merge_and_impute.py   # menghasilkan data_final_imputed.json
3. cd dataviz-react && npm install && npm run dev   # menjalankan visualisasi pada http://localhost:5173
